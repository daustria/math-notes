Contour Integration¶

A quick discussion on contour integrals and their light on holomorphic functions.

Note

A domain \(D\) will refer to an open connected subset of the complex plane. Functions will usually have the form \(f: D \to \mathbb{C}\) for some domain D.

Our first theorem will connect the presence of an antiderivative with path independence and vanishing loop integrals. In fact these are equivalent in some sense.

Theorem

Let \(f\) be continuous in a domain D. The following are equivalent:

  1. \(f\) has an antiderivative \(F\) in D

  2. If \(\Gamma\) is a loop contour, \(\int_{\Gamma} f(z)\,dz=0\).

  3. Contour integrals of \(f\) are path independent.

We will prove (sketch) \(1 \implies 2 \implies 3 \implies 1\).

\(1 \implies 2\)

We can use a version of the fundamental theorem of calculus. For the sake of presentation assume that \(\Gamma\) is a directed smooth curve (if not, run the proof on each of its components). Hence its parameterization \(z(t) : [a,b] \to \mathbb{C}\) is continuously differentiable. By the chain rule we have

\[\frac{d}{dt} F(z(t)) = f(z(t))z'(t)\]

and so

\[\int_{\Gamma}f(z)\,dz = \int_a^b f(z(t))z'(t)\,dt = F(z(b)) - F(z(a)).\]

where the last equation is just a small extension to the fundamental theorem of calculus for real valued functions, easily proven. Since \(z(b)=z(a)\) the above equation is 0, which is what we needed to show.

\(2 \implies 3\)

Let \(z,w\) be two points in our domain. Let \(\gamma_1, \gamma_2\) be two distinct paths (contours) from \(z\) to \(w\), entirely contained in \(D\). Let \(\Gamma\) be the contour formed by appending \(-\gamma_2\) to \(\gamma_1\). Then by hypothesis the integral of \(f\) over \(\Gamma\) vanishes. This is exactly what we want since

\[0 = \int_{\Gamma} f = \int_{\gamma_1}f - \int_{\gamma_2}f\]

\(3 \implies 1\)

Fix a point \(x\) in \(D\) and define \(F(z) = \int_x^z f(s)\,ds\) where the notation indicates integrating over a straight line from \(x\) to \(z\). We must show that \(F\) is differentiable and its derivative is \(f\), which can be done directly. By path independence, we have

\[F(z+w) - F(z) = \int_x^{z+w} f - \int_x^z f = \int_{z}^{z+w} f\]

So the proof is done if we can show

\[\frac{1}{w} \int_{z}^{z+w} f(s)\,ds \to f(z)\]

as \(w \to 0\). Parameterize the contour as \(z+wt\) for \(0 \leq t \leq 1\). Then the integral becomes

\[\frac{1}{w} \int_0^1 f(z+wt)w\,dt = \int_0^1 f(z+wt)\,dt.\]

Finally we have

\[\int_0^1 f(z+wt)\,dt - f(z) = \int_0^1 f(z+wt)-f(z)\,dt\]

which tends to \(0\) as \(w \to 0\) by continuity of \(f\) \(\blacksquare\).

How do we check if all loop integrals vanish? One tool we can use is the deformation invariance theorem.

Theorem (Deformation Invariance)

Let \(f\) be analytic in a domain D. If \(\Gamma_0, \Gamma_1 \subset D\) are two loops such that \(\Gamma_0\) can be continuously deformed into \(\Gamma_1\) then

\[\int_{\Gamma_0} f(z)\,dz = \int_{\Gamma_1}f(z)\,dz.\]

Hint

By continuously deformed, we mean that there is a continuous function \(z(s,t):[0,1] \times [0,1]\) such that \(z(0,t)\) parameterizes \(\Gamma_0\) and \(z(1,t)\) parameterizes \(\Gamma_1\).

For ease of presentation we assume that \(f'\) is continuous, and \(z(s,t)\) is \(C^2\). Define

\[I(s) = \int_{\Gamma_s} f(z)\,dz = \int_0^1 f(z(s,t)) \frac{\partial z(s,t)}{\partial t}\,dt.\]

We differentiate \(I\) with respect to \(s\) using Leibniz rule, justified by the fact that the integrand is \(C^1\) as a function of \(s\). We have, by chain rule,

\[\frac{dI(s)}{ds} = \int_0^1 f'(z(s,t)) \frac{\partial z}{\partial s} \frac{\partial z}{\partial t} + f(z(s,t)) \frac{\partial^2 z}{\partial s \partial t}\,dt.\]

Notice that because \(z\) is \(C^2\) the integrand above is also the same expression as if we differentiated with respect to t. Thus

\[\frac{dI(s)}{ds} = \int_0^1 \frac{\partial}{\partial t} f(z(s,t))\frac{\partial z}{\partial s}\,dt.\]

Now invoke the fundamental theorem of calculus, and note that the resulting expression vanishes because \(z(s,1) = z(s,0)\). In particular \(I\) is constant, and so \(I(0) = I(1)\) as required \(\blacksquare\).

If a domain \(D\) is such that every loop integral can be continuously deformed to a point, then all loop integrals of \(f\) vanish. This we know now is equivalent to the existance of an an antiderivative for \(f\) and path independence of its contour integrals. Such a domain \(D\) is called simply connected if this is true, and is usually the case when \(D\) has no holes. The loop integrals vanishing is the content of Cauchy’s integral theorem, but keep in mind the other two conditions as well.

Cauchy’s Integral Theorem

If \(f\) is analytic in a simply connected domain \(D\) and \(\Gamma\) is a loop then \(\int_{\Gamma}f = 0\).

Now consider when \(f\) is analytic on a simply connected domain \(D\), and we want to integrate \(f\) over some closed contour, but it has a singularity inside the contour, like \(\frac{1}{z}\) on \(|z|<1\). In this setting, the above conditions do not apply. Also, many texts establish early on that when \(C\) is a circle centred at \(a\) of some radius we have \(\int_C \frac{dz}{z-a}\) is \(2 \pi i\) when \(a\) lies inside \(C\) and \(0\) otherwise.

We can handle such cases with Cauchy’s integral formula.

Cauchy’s integral formula.

Let \(f\) be analytic in a simply connected domain \(D\) containing a simple, closed, positively oriented contour \(\Gamma\). If \(a\) is any point inside \(\Gamma\) then

\[f(a) = \frac{1}{2\pi i} \int_{\Gamma} \frac{f(z)}{z-a}\,dz.\]

Using the deformation invariance theorem, we may shrink the contour \(\Gamma\) to a positively oriented circle \(C_r\) centred at \(a\) with a radius \(r\) small enough so that the circle stays inside \(D\). We rewrite the integral on the right hand side to be

\[\oint \frac{f(z)}{z-a}\,dz = \oint \frac{f(a)}{z-a}\,dz + \oint \frac{f(z) - f(a)}{z-a}\,dz.\]

The first integral is known to be \(2 \pi i f(a)\), so the proof is done if we can show that the second integral approaches \(0\) as \(r \to 0\). By analyticity, we have that

\[\frac{f(z) - f(a)}{z-a} = f'(a) + \phi(z)\]

where \(\phi(z)\) is a function satisfying \(\lim_{z \to a} \phi(z) = \phi(a) = 0\). Choose \(r\) so small so that \(|\phi(z)| < 1\) for each \(z\) inside and on \(C_r\). Then we have

\[\bigg| \oint \frac{f(z)-f(a)}{z-a}\,dz \bigg| \leq 2 \pi r (|f'(a)| + 1).\]

which tends to \(0\) as \(r\) does, completing the proof \(\blacksquare\).

Note that this theorem is also true if our contour is a loop with self intersections. Just separate the contour into multiple loops with winding number one. Also this result generalizes further to higher derivatives. The formula formally obtained from differentiating under the integral sign turns out to be correct.

Cauchy’s Integral Formula

Let \(f\) be analytic in a simply connected domain \(D\) containing a simple, closed, positively oriented contour \(\Gamma\). If \(a\) is any point inside \(\Gamma\) then

\[f^{(n)}(a) = \frac{n!}{2 \pi i}\int_{\Gamma} \frac{f(s)}{(s-a)^{n+1}}ds\]

for each \(n \geq 1\).

We will prove a more general result. Notice in the statement that \(g\) is just continuous and our contour \(\Gamma\) is just a contour, not necessarily closed or positively oriented.

Lemma

Let \(g\) be continuous on some contour \(\Gamma\). For \(a\) not on \(\Gamma\) define

\[G(z) := \int_{\Gamma} \frac{g(z)}{z-a}\,dz\]

Then \(G(z)\) is analytic at each point not on \(\Gamma\). Its derivative is given by

\[G'(z) = \int_{\Gamma} \frac{g(z)}{(z-a)^2}\,dz.\]

Define \(J = J(z)\) to be the quantity

\[J(z) = \frac{G(z) - G(a)}{z-a} - \int_{\Gamma} \frac{g(\zeta)}{(\zeta - a)^2}\,d\zeta.\]

The proof is done if we can show that \(J \to 0\) as \(z \to a\). Using some algebra we have

\[\begin{split}\begin{align} J(z) &= \int_{\Gamma} \frac{g(\zeta)}{(\zeta - z)(\zeta - a)}\,d\zeta - \int_{\Gamma} \frac{g(\zeta)}{(\zeta - a)^2}\,d\zeta \\ &= (z-a) \int_{\Gamma} \frac{g(\zeta)}{(\zeta - z)(\zeta - a)^2}\,d\zeta \end{align}\end{split}\]

where we assume that \(z\) is in a small enough neighbourhood of \(a\) so that \(z\) does not lie on \(\Gamma\). The integrand is continuous on the image of \(\Gamma\) which is a compact set. Hence it achieves its maximum value there, call it \(M\). This lets us bound \(|J(z)|\) by \(|z-a| M \ell(\Gamma)\) which tends to \(0\) as \(z \to a\). \(\blacksquare\).

The technique applied here can be generalized, to prove the validity of computing \(G^{(n+1)}(z)\) from \(G^{(n)}(z)\) by differentiating under the integral sign, but we will not show those details here.

This result is enough to prove the general version of Cauchy’s integral formula. Just choose \(\Gamma\) to be a circle centred at \(a\) of small enough radius so that \(f\) is analytic inside and on the circle. This also proves that if \(f\) is analytic on \(D\), then so is \(f^{(n)}\) for each \(n \geq 1\) \(\blacksquare\).

Thus \(f\) being differentiable once implies differentiability of all orders. So if \(f\) has an antiderivative then \(f\) itself is analytic. Recall from the first theorem that the existance of an antiderivative is equivalent to two other conditions relating to contour integrals of \(f\). This is basically the content of Morera’s Theorem which we now state.

Morera’s Theorem

Let \(f\) be continuous in a domain \(D\). Suppose that all loop integrals vanish. Then \(f\) is analytic on \(D\).

Note we also could have also used path independence of contour integrals of \(f\) \(\blacksquare\).
