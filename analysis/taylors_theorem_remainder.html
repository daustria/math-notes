Taylor’s Theorem with Remainder¶

We discuss a version of Taylor’s theorem that uses only first order partial derivatives.

Theorem

Let \(g:\mathbb{R}^n \to \mathbb{R}\) be differentiable on an open set \(U \subset \mathbb{R}^n\) starshaped with respect to a point \(a \in U\). There exists a \(t_0\) with \(0 < t_0 < 1\) such that

\[g(x) = g(a) + \sum _{i=1}^n \frac {\partial g}{\partial x_i} (a + t_0(x-a)) (x^i - a^i).\]
