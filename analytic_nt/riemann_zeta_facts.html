On the Riemann-Zeta function¶

The zeta function is defined as

\[\zeta(s) = \sum_{n \geq 1} n^{-s},\quad \sigma = \Re(s) > 1.\]

Proposition

On \(\sigma\) > 1 we have

\[\zeta(s) = \frac{s}{s-1} + s\int_1^{\infty} \frac{\{x\}}{x^{s+1}}\,dx\]

where the integral on the right hand side is an analytic function for \(\sigma > 0\).

This yields an analytic continuation for \(\zeta(s)\) for the region \(\sigma > 0,\, s \neq 1\). This also shows \(\zeta(s)\) has a simple pole at \(s=1\) of residue \(\lim_{s\to 1^+}(s-1)\zeta(s) = 1\). Also with this formula one may see that \(\zeta(s) \neq 0\) on \(\sigma = 1\), for example by noting that the real part is positive.

Apply this formula with \(a_n=1\) to obtain that, for \(\sigma > 1\), we have

\[\begin{split}\begin{align} \zeta(s) &= s \int_1^\infty \frac{[x]}{x^{s+1}}\,dx \\ &= s \int_1^\infty x^{-s}\,dx - s \int_1^\infty \frac{\{x\}}{x^{s+1}}\,dx \\ &= \frac{s}{s-1} - s \int_1^\infty \frac{\{x\}}{x^{s+1}}\,dx . \end{align}\end{split}\]

where we used that \([x] = x - \{ x \}\). To see that the integral is analytic on \(\sigma > 0\) one may apply the Cauchy-Riemann equations, applying Leibniz rule to differentiate the real and imaginary parts with respect to \(\sigma\) and \(t\). Notice that the integrand is discontinuous only on integers, so Riemann integrability applies. One might also apply Morera’s Theorem with Fubini \(\blacksquare\).

Proposition

\(\zeta(s) \neq 0\) for \(\sigma = \Re(s) \geq 1\).

The proof for \(\sigma > 1\) is an easy application of the triangle inequality, so we discuss only \(\sigma = 1, s \neq 1\).

Lemma

For \(\sigma > 1, t \in \mathbb{R}\), we have

\[\Re \log \zeta(\sigma + it) = \sum_{n \geq 2} \frac{\Lambda(n)}{n^\sigma \log n} \cos (t \log n).\]

We have, for \(s = \sigma + it\),

\[\log \zeta(s) = \log \prod_p ( 1 - p^{-s} )^{-1} = \sum_p -\log (1-p^{-s})\]

Where the index is over unique primes. Now we apply the Taylor series identity

\[-\log(1-x) = \sum_{n \geq 1} \frac{x^n}{n}\]

to obtain

\[\log \zeta(s) = \sum_p \sum_{n \geq 1} \frac{1}{np^{ns}}.\]

Notice this summation is absolutely convergent because \(|\log \zeta(s)| \leq \log \zeta(\sigma) < \infty\). So instead we can write the summation over prime powers,

\[\log \zeta(s) = \sum_{x=p^n, n \geq 1} \frac{\log p}{\log x} \frac{1}{x^{ns}} = \sum_{n \geq 2} \frac{\Lambda(n)}{n^s \log n}.\]

Note the von Mangoldt function \(\Lambda(n)\) vanishes on non-prime powers, allowing us to write the summation over all \(n \geq 2\). Comparing real parts of this identity completes the proof \(\blacksquare\).

Lemma

For \(\sigma > 1, t \in \mathbb{R}\) we have

\[|\zeta(\sigma)^3 \zeta(\sigma + it)^4 \zeta(\sigma + 2it)| \geq 1.\]

Consequently, \(\zeta(1+it) \neq 0\) for any \(t\) by a simple continuity argument.

Taking the logarithm of both sides, it suffices to prove

\[\Re (3 \log \zeta(\sigma) + 4 \log \zeta(\sigma + it) + \log \zeta(\sigma + 2it)) \geq 0.\]

Using the previous lemma, this reads as

\[\sum_{n \geq 2} \frac{\Lambda(n)}{n^s \log n} (3 + 4 \cos (t \log n) + \cos (2t \log n)) \geq 0.\]

Notice that the term on the left is positive, so we would be done if the function \(3 + 4 \cos x + \cos 2x\) is non-negative. This can be proven easily with calculus. We find the minimum by looking at where the derivative vanishes,

\[-4 \sin x - 2 \sin 2x = 0 \iff -\sin x = \sin x \cos x \iff x = \pi + 2\pi \mathbb{Z}.\]

But at \(x = \pi\), we have \(4 \cos x + \cos 2x = -3\). Hence \(3 + 4 \cos x + \cos 2x \geq 0\). This completes the proof of this lemma and also the proposition \(\blacksquare\).

Proposition

\(-\frac{\zeta'}{\zeta}(s)\) is analytic on \(\sigma \geq 1\) except for a simple pole at \(s=1\) with residue \(1\).

By analytic continuation, both \(\zeta\) and \(\zeta'\) can be taken to be analytic on \(\sigma > 0, s \neq 1\). Because \(\zeta(s) \neq 0\) for \(\sigma \geq 1\), \(-\frac{\zeta'}{\zeta}(s)\) is analytic on \(\sigma \geq 1, s \neq 1\).

Write \(\zeta(s)(s-1) = sf(s)\) for \(f\) analytic on \(\sigma > 0\). Differentiating both sides and rearranging yields

\[-(s-1)\frac{\zeta'}{\zeta}(s) = -\frac{f'(s)}{\zeta(s)} - \frac{f(s)}{\zeta(s)} + 1.\]

Now take \(\lim_{s \to 1^+}\), the right hand side resolves to 1 because \(\zeta(s)\) blows up \(\blacksquare\).

Proposition

On \(\sigma > 1\), we have the following Dirichlet series representation,

\[-\frac{\zeta '}{\zeta}(s) = D(\Lambda, s) = \sum_{n \geq 1} \frac{\Lambda(n)}{n^s}.\]

where \(-\zeta'(s) = \sum_{n \geq 1} (\log n) n^{-s}\).

We write the proposition in the form

\[D(\Lambda, s) D(1, s) = -\zeta'(s)\]

By absolute convergence on \(\sigma > 1\), the product on the left hand side converges to the Cauchy product,

\[\sum_{n \geq 1}\sum_{d|n} \frac{\Lambda(d)}{n^s}\]

It suffices to show that the coefficients of this Dirichlet series evalutes to \(\log n\) for each \(n \geq 1\). This is straightforward,

\[\sum_{d|n} \Lambda (d) = \sum_{p^a|n} \log p = \sum_{p|n} e \log p = \log p_1^{e_1} \ldots p_n^{e_n} = \log n\]

where summations are over prime powers or distinct primes dividing \(n\) \(\blacksquare\).
