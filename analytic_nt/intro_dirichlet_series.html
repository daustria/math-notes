Primer on Dirichlet Series¶

Arithmetic Functions¶

Arithmetic functions have the form \(f: \mathbb{N} \to \mathbb{C}\). They are additive if \(f(m+n)=f(m)+f(n)\) for coprime \(m,n\) and completely additive if the rule holds for all \(m,n\). Same rule for multiplicative but for the product \(mn\).

\(\nu(n)\) is the distinct number of prime divisors. This is completely multiplicative.

\(\Omega(n)\) is the number of prime divisors not necessarily distinct. Also multiplicative.

\(d(n) = \sum_{d|n} n\) is the divisor function. This is multiplicative, since any divisor of \(mn\) can be split into two relatively prime parts that are divisors for \(m\) and \(n\) alone. An interesting approximation is this that,

Proposition

for any \(\epsilon>0\) we have \(d(n) = O(n^{\epsilon})\).

This result can be improved, but we don’t want to go too far.

The case that \(d(n) = O(\sqrt{n})\) is pretty easy, just notice that for any pair whose product is \(n\), one of the partners is at most \(\sqrt{n}\).

To tackle the general case we can ‘localize’ the problem to the prime divisors of \(n = p_1^{a_1}\ldots p_k^{a_k}\). We fix an \(\epsilon>0\) and a prime \(p=p_j\) dividing \(n\) with power \(a=a_j\). We will find a constant \(C>0\) such that

\[\frac{d(p)}{p^{a \epsilon}} = \frac{a+1}{p^{a \epsilon}} \leq C\]

where the constant \(C\) depends only on \(\epsilon\). It suffices to do this since \(d\) is multiplicative, we can later take the product of all such \(C\) to bound \(d(n)/n^{\epsilon}\).

We split our work into cases. If \(p > e^{1/\epsilon}\) then \(p^{\epsilon} > e\), so

\[\frac{a+1}{p^{a \epsilon}} \leq \bigg( \frac{e}{p^\epsilon} \bigg)^a \leq 1\]

where we used that \(e^a \geq a+1\) using the Taylor series expansion of \(e^x\). For all the other, smaller primes, we have

\[\frac{a+1}{p^{a \epsilon}} \leq \frac{a+1}{2^{a \epsilon}} \to 0\]

as \(a \to \infty\). Thus we may find a positive constant that bounds this expression uniformly for all \(a \geq 0\). We set our constant \(C\) to be the maximum of this number and 1 \(\blacksquare\).

\(\mu(n)\) is the Mobius function. It has \(\mu(1)=1\) and

\[\begin{split}\mu(n) = \begin{cases} (-1)^{\nu(n)} & \text{n squarefree} \\ 0 & \text{otherwise.} \end{cases}\end{split}\]

It can be shown rather easily that \(\sum_{d|n} \mu(d) = 0\) with the exception that \(\mu(1)=1\). This can be applied to show the Mobius inversion formula,

Theorem

If \(f,g\) are arithmetic functions then

\[f(n) = \sum_{d|n} g(n) \quad \forall n \iff g(n) = \sum_{d|n}\mu(d)f(n/d) \quad \forall n.\]

Euler phi \(\phi(n)\) is defined by

\[\phi(n) = n \prod_{p|n}\bigg( 1 - \frac{1}{p} \bigg) = |\mathbb{Z}_n^*|\]

where the product is over distinct prime divisors. I like the probabilistic interpretation of this function gained by writing

\[\frac{\phi(n)}{n} = \prod_{p|n} \bigg( 1 - \frac{1}{p} \bigg)\]

so that probability of an integer in \(1,\ldots,n\) being prime to \(n\) is the same probability that it was not divisible by any prime \(p\) in the factorization of \(n\).

The von Mangoldt function \(\Lambda(p^a) = \log p\) for prime powers, and \(0\) otherwise. It is not additive or multiplicative but will be important in the Prime Number Theorem.

Partial summation is a useful technique when dealing with arithmetic functions and series.

Theorem

Let \(a_n\) be a sequence of complex numbers and \(f\) be \(C^1\) on \([1,x]\). Define \(A(x) = \sum_{n \leq x} a_n\). Then we have

\[\sum_{n \leq x} a_n f(n) = A(x)f(x) + \int_1^x A(t)f'(t)\,dt.\]

Formal Dirichlet Series¶

The formal Dirichlet series for an arithmetic function \(f\) is given by

\[D(f,s) = \sum_{n \geq 1} f(n)n^{-s}.\]

Sum is defined in usual way. Product is defined as

\[D(f,s)D(g,s) = \sum_{n \geq 1} h(n)n^{-s} = \sum_{n \geq 1}\sum_{de=n}f(d)g(e)n^{-s}.\]

When \(f\) is multiplicative notice that unique prime factorization yields

\[D(f,s) = \prod_p \sum_{\nu \geq 0} \bigg( f(p^{\nu}) p^{-\nu s} \bigg).\]

An important series is the Riemann-zeta function,

\[\zeta(s) = \sum_{n \geq 1} n^{-s} = D(1,s) = \prod_p \frac{1}{1 - p^{-s}}\]

Then

\[D(\mu, s) = \frac{1}{\zeta(s)}.\]

This can be proved from that identity involving \(\sum_{d|n} \mu(d)\) we saw earlier.

Now we discuss analyticity of Dirichlet functions. Note that in this context we use \(s\) to denote complex variables and often write \(s = \sigma + it\).

Theorem

In the setting of partial summation, suppose that for \(A(x) = O(x^{\delta})\),

For \(\Re(s)> \delta\) we have

\[\sum_{n \geq 1} \frac{a_n}{n^s} = s \int_1^\infty \frac{A(t)}{t^{s+1}}\,dt.\]

Simply apply the partial summation theorem with \(f(x) = x^{-s}\) and we get

\[\sum_{n \leq x} \frac{a_n}{n^s} = A(x)x^{-s} + s \int_1^x \frac{A(t)}{t^{s+1}}\,dt.\]

Taking \(x \to \infty\) yields the desired result. Note that the \(A(x)x^{-s}=O(x^{\delta - \Re(s)})\) which will vanish in the limit. Here we have \(f: \mathbb{R} \to \mathbb{C}\), so one might wonder how we got the above identity. Simply write \(f(s) = u(s) + i v(s)\), apply partial summation for \(u,v\) separately, and then add them up \(\blacksquare\).
