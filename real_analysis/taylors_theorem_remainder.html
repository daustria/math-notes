Taylor’s Theorem with Remainder¶

We discuss a multivariable version of Taylor’s theorem that uses only first order partial derivatives.

Theorem

Let \(g:\mathbb{R}^n \to \mathbb{R}\) be differentiable on an open set \(U \subset \mathbb{R}^n\) star shaped with respect to a point \(a \in U\). There exists a \(t_0\) with \(0 < t_0 < 1\) such that

\[g(x) = g(a) + \sum _{i=1}^n \frac {\partial g}{\partial x_i} (a + t_0(x-a)) (x^i - a^i).\]

Since \(U\) is star shaped about \(a\) then then \(g\) is differentiable on the line \(X(t) = a + t(x-a)\) for any \(x\) in \(U\). In other words the function \(g(X(t))\) is differentiable on \((0,1)\). Apply the mean value theorem (single variable) and chain rule (multivariable) to \(g(X(t))\) and we obtain the equality stated in the theorem \(\blacksquare\).

An immediate corollary is some bounds given in the following form

Corollary

Let \(g,a,U\) be as before. If \(\frac{\partial g}{\partial x_i} < K\) on \(U\) then

\[|g(x) - g(a)| < K \sqrt{n} ||x-a||,\quad x \in U.\]

More generally if \(G(x) = (g_1(x), \ldots g_m(x))\) is differentiable on \(U\) starshaped with respect to a point \(a\) and \(\frac{\partial g_i}{\partial x_j} < K\) on \(U\) then

\[||G(x) - G(a)|| < K \sqrt{nm} ||x-a||, \quad x \in U.\]

The proof is simple, uses the triangle inequality applied to our version of Taylor’s theorem \(\blacksquare\).

Also, supposedly there is a way to use this theorem to prove the interchangability of partial derivatives, but I don’t know how to show it yet.

Corollary

If \(g\) is \(C^r\) on \(U\) then partial derivatives of \(g\) of up to order \(r\) is independent of the order. Namely if \((i_1,\ldots,i_k)\) and \((j_1,\ldots,j_k)\) are permutations with \(k \leq r\) then

\[\frac{\partial^k g}{\partial x_{i_1} \ldots x_{i_k}} = \frac{\partial^k g}{\partial x_{j_1} \ldots x_{j_k}}.\]

There is another version of this theorem that deals with smooth functions.

Theorem

Let \(g,U,a\) be as before. Then there are \(C^{\infty}\) functions \(g_i\) on \(U\) such that

\[g(x) = g(a) + \sum _{i=1}^n g_i(x)(x^i - a^i).\]

The \(g_i\) satisfy \(g_i(p) = \frac{\partial g}{\partial x_i}(p)\). In fact, the \(g_i\) have the form

\[g_i(x) = \int_0^1 \frac{\partial g}{\partial x_i}(a + t(x-a))\,dt\]

The smoothness of the \(g_i\) can be shown directly from the smoothness of \(\frac{\partial g}{\partial x_i}\). The partial derivatives of \(g_i\) can be computed using Leibniz rule. For example, view the integrand in the right hand side of \(g_i\) as a smooth function \(F(x,t)\) on \(U \times [0,1]\). Then

\[\frac{\partial g_i}{\partial x_j} = \frac{\partial}{\partial x_j} \int_0^1 F(x,t)\,dt = \int_0^1 \frac{\partial}{\partial x_j} F(x,t)\,dt.\]

Furthermore the integrand on the right hand side is smooth, from smoothness of \(g\). In general, we can compute partial derivatives of \(g_i\) of arbitrary order by repeatedly applying Leibniz rule, to obtain a function of the form \(\int_0^1 F(x,t)\,dt\) where \(F\) is smooth on its domain. So it suffices to show that such a function is continuous, and has continuous partial derivatives.

To obtain the existence of the \(g_i\), we perform a similar computation as the previous theorem.

\[g(x) - g(a) = \int_0^1 \frac{d}{dt} g(a + t(x-a)) = \sum_{i=1}^n (x^i - a^i) \int_0^1 \frac{\partial g}{\partial x_i}(a + t(x-a))\,dt\]

which completes the proof \(\blacksquare\).

A connection between these two theorems lies in the mean value theorem for integrals. Since \(f_i = \frac{\partial g}{\partial x_i}(a + t(x-a))\) is differentiable as a function of \(t\), there is a point \(0<t_0<1\) such that \(\frac{1}{1-0} \int_0^1 f_i(t) = f_i(t_0)\)

This theorem is quite helpful in the study of differential topology. For example, it can be used to show that the vector space of tangent vectors of an \(n\) dimensional smooth manifold \(M\) is isomorphic to \(\mathbb{R}^n\) when the tangent vectors are defined as germs acting on \(C^{\infty}(M)\), the space of smooth real valued functions \(f: M \to \mathbb{R}\) on \(M\).
